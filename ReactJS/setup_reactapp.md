# How to set up the React project or  How to install the react to your project?

#### Use the following command in the sequence:

``` 
npx create-react-app withreact

cd withreact

npm start

npm install --save-dev ajv@^8 

```

OR(using vite)

```
npm create vite@latest

npm i

npm run dev

```
